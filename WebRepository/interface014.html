JGoodies focuses on Java look, UI design and usability. We provide articles, libraries, example applications, desktop patterns and a Swing application architecture.

:: Building Elegant Swing Screens ::

Our free professional open source desktop libraries help you build well designed Swing forms and screens. Test our free tools and demos and see yourself.

:: Working With Swing Efficiently ::

The Swing Suite product helps you create sophisticated, professional and rich Swing UIs. It is built upon our free libraries and brings all sources for our tools and demos.

The Suite demonstrates Java desktop patterns, shows how to tie things together, how to build elegant apps efficiently, and helps you get started with Swing quickly.
