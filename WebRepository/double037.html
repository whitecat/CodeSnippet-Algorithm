Starting out with Java

Converting type double to type int.
faq269-1093
Posted: 12 Sep 01

To convert a double to an int you need to use a cast like this one:

int i = (int)d;

Here is a simple Java program that uses a cast to convert a double to an int:

public class DoubleToInt {

    public static void main(String[] args) {
        double d = 10.5;
        int i = (int)d;
        System.out.println(i);
    }
}
