Downloads

The main GlassFish downloads are listed below. Stay informed through the Aquarium and GlassFish For Business. For additional downloads, see the build archive.


GlassFish Server
 The GlassFish Server is available in different releases to address different requirements.
For more details, check out this detailed comparison between v2 and v3.
GlassFish v2.1.1 A final release based on the Java EE 5 standard, with clustering, load balancing, high availability, Update Center, and support for SailFin (Telco App Server) and AIX 6.1. Commercially supported by Sun and recommended for production environments.
Community Release | Sun's Supported Release | Quick Start Guide | Documentation | Support | Training
GlassFish v3 A final release implementing the Java EE 6 standard. Includes an extensible core based on OSGi, Admin Console and Update Center. It does not have HA, clustering and other features.
Community Release | Quick Start Guide | Installation Guide | Documentation | Support | Patches
Tools Maximize developer productivity with these NetBeans and Eclipse bundles containing GlassFish v2 and v3.
GlassFish Tools Bundle for NetBeans Experience the industry's best Java EE developer experience with NetBeans and GlassFish.
Download | Documentation (Web App) |  Documentation (EJB and Web Services) | Release Notes | Installation Instructions
GlassFish Tools Bundle for Eclipse An easy-to-install, out-of-the-box Java EE developer experience for GlassFish users who prefer the Eclipse IDE.
Download | Documentation | Release Notes
Other
 These bundles and artifacts provide complete solutions for your business integration, content aggregate, telco server, and continuous integration needs.
GlassFish v3.1 An early access release that will eventually replace v3.0.
Download Nightly
GlassFish ESB v2 Project OpenESB, based on GlassFish v2, is an Enterprise Service Bus (ESB) runtime, based on Java Business Integration (JBI).
More Info | Download | Documentation | Builds | Additional Components
GlassFish Web Space Server GlassFish Web Space Server, based on GlassFish v2 and Liferay, is a next-generation portal server platform.
Download | Documentation | Support | Training | Blogs | Community
SailFin SailFin SIP Server, based on GlassFish v2.1, is a platform for developing and deploying converged Java EE and SIP servlet applications.
More Info | Documentation | Milestone Build | Promoted Build | Nightly Build
