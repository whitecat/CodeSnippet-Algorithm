Java console - public options

David McMullin
January 29th, 2010
Filed under: NetBackup, Features, Backup and Archiving
3 Agree, 0 Disagree

There have been ideas to allow saving filter and sorts, or screen formats (column order, etc)

Can we get an option to make these public or shared?

I have had issues in the past where a user had hidden a column or it was not displayed the same as mine, and it made trouble shooting more arduous.

If we had shared options, we could standardize the displays, or even choose to switch to the OPS team views while working with them, then easily switch back to my admin view.
