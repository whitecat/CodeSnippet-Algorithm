Final Cut Server: Upgrade Java for Mac OS X on computers using Final Cut Server client

  • Last Modified: September 24, 2008
  • Article: HT3166

Summary

If you are running Final Cut Server client software on Apple computers, install the latest version of Java for Mac OS X.

Products Affected

Java, Final Cut Server 1.x

Follow these steps to install the latest version of Java for Mac OS X using Software Update.

  1. Open Software Update by choosing Software Update from the Apple menu.
  2. From the list of updates that appears, select only the checkbox for Java for Mac OS X Update.
    • For Mac OS X v10.4 Tiger, select Java for Mac OS X 10.4, Release 7.
    • For Mac OS X v10.5 Leopard, select Java for Mac OS X 10.5 Update 2.
  3. Click Install.

You can also manually download updates to Java for Mac OS X from Apple Downloads.

Additional Information

View the Technical Specifications for Final Cut Server client software for Mac OS X.
