On October 30, J2SE 5.0 will reach the end of its service life (EOL). This means that updates and security fixes will no longer be available on a regular basis via the Java websites. However, you can preserve and extend your Java investment with fast access to critical fixes and a much longer support roadmap (out to 2015 and beyond) by subscribing to Java for Business (JFB).

The JFB program offers ongoing access to the regular security releases Sun creates. JFB also enables you to engage with Sun's Java experts to help resolve any problems that may arise. The Sun Java experts continue to expand support for new Operating Systems and browsers over time so that you can continue to use your J2SE 5.0 applications even as your underlying environment evolves. JFB also offers enterprise-level services, tools, tips and access to our world-renowned Java experts.

Take your Java performance to the max. Increase the return on the investment you made in your J2SE 5.0 application by keeping it safe, current with the latest platforms like Windows 7 and IE 8, and able to file bugs and receive fixes for up to 15 years.

We know that building a serious business around Java requires confidence, expertise, and sometimes a little help. For more information, visit us at: http://www.sun.com/software/javaforbusiness/ or contact us.
