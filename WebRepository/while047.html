Java: Changing while loop to for loop
I'm trying to change this while loop in to a for loop:

Code:
public Product findProduct(String name)
{
int maxID = stock.length;
int i = 0;
boolean notFound = true;
Product temp = null;
while (i < maxID && notFound) {
if ((stock[i] != null) && (stock[i].getName() == name)) {
notFound = false;
temp = stock[i];
}
}
return(temp);
}


This is what I have so far:


Code:
for(int i = 0; i< maxID && notFound; i++)
{
if ((stock[i] != null) && (stock[i].getName() == name))
{
notFound = false;
temp = stock[i];
}
}


Unfortunately, I'm getting a message that it can't find the variable maxID. Any help would be much appreciated.

One possible solution
If I understand what you are trying to do here, I think something like this should work:

Code:
    public Product findProduct(String name)
    {
        for( int i = 0; i < stock.length; i++ )
        {
            if( stock[i].getName().equals( name ) )
                return stock[i];     // found it
        }

        return null;    // didn't find it, just return "null"
    }


Please note an important change that I made:

Code:
if( stock[i].getName().equals( name ) )


Rather than:
Code:
if( stock[i].getName() == name )


When comparing Strings you want to check for if the TEXT INSIDE the Strings are equal, not if the String objects themselves are equal. This is an important distinction.

Hope this helps!
