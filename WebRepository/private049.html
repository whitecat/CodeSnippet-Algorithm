Export a private key from a Java keystore

While trying to install my SSL certificate in JBoss one approach was to combine all your certificates into a PKCS12 keystore. To do this you need to export the private key from your JKS keystore, and then use openssl to export your chained certificates into a new keystore.

The links on Atomeo are dead, and the cached copy of the code threw some errors with JDE 1.6.Â  Eventually I stumbled across a post by Anand Rajasekar containing ExportPrivateKey.zip which can be used to export the private key from a JKS keystore.

The syntax is:

java -jar ExportPrivateKey.zip {keystore_path} JKS {keystore_password} {alias} {target_file}
To export the private key from web.keystore for a certificate with the alias bas to the file bas.key you'd run:
java -jar ExportPrivateKey.zip web.keystore JKS KEYSTORE_PASSWORD bas bas.key
