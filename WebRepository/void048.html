partnumber is declared as an instance variable, thus it cannot be accessed from static methods, so to resolve this problem you need to declare this variable as static
import java.util.*;

public class ProjectOne
{
	 
	static double partNumber = 0;

	public static void main(String[] args)
	{			
		Scanner input = new Scanner(System.in);
		partNumber = input.nextDouble();
		partNumber = partNumber + 12345;
		Display();
	}

	public static void Display()
	{
		System.out.println("This is the new part number" + partNumber);
	}
}
