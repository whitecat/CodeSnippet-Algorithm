Java for Chrome

Google Chrome requires Java 6 Update 12 or greater (Java 6u12+), as indicated in the Google Chrome Help Center. Although the Chrome browser is not officially supported by Sun for Java 6, we at Sun are working hard to add formal support for Chrome as soon as possible, so please check back for updates.
