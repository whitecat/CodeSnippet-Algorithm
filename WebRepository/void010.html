Java - Void Function

Expert: Brian Ngure - 9/26/2007

Question
Hey Brian, I am just beginning to learn Java and was wondering how you would go about doing the following problem:

Write a method void recordGame(boolean won) which updates the number of games played and (if appropriate) the number of games they won.

I do not know how to write the body of the function, given that it is a void (thus no returns) when it needs to update two variables (gamesWon and gamesPlayed). I have tried using if else statements, but none that I have tried can compile

Answer
Hi,

It would help if you could post the code you have written so far.

Here is an example.


private int noOfGamesPlayed = 0;
private int noOfGamesWon = 0;

private void recordGame(boolean won) {
   noOfGamesPlayed++; // Add 1 to the number of games played

   if(won) { // If "won" is 'true', add 1 to the number of games won
       noOfGamesWon++;
   }
}
