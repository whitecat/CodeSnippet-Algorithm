How to convert a String to int in Java

Member

convert a String to int in Java

It is always tricky to convert from one variable type to another. This HowTo explains how to convert a String to an int in the Java programming language.
Using this technic even beginners can easily convert from a String to an int. If you are a very good, seasoned programmer you probably don't need to read this How-To. Instead, you can skip to the last step where there is the surce code all in an overview.

Difficulty: Easy
Instructions

Things You'll Need:

  • A computer
  • A java compiler
  • A text editor
  1. Step 1

    Declare the String:
    String theString;

  2. Step 2

    Give the String a value:
    theString = "69";

  3. Step 3

    Declare the int:
    int theInt;

  4. Step 4

    Use the parseInt method of Integer to convert the String to an int value and use the result to fill the variable theInt:
    theInt = Integer.parseInt(theString);

  5. Step 5

    After you did that you can check if it worked. Use a println to print it to both, the old String and the new int to the console:

    System.out.println(theInt);
    System.out.println(theString);

    Both should look the same. In our case it should print the following:
    69
    69

  6. Step 6

    Complete Listing:

    String theString;
    theString = "69";
    int theInt;
    theInt = Integer.parseInt(theString);
    System.out.println(theInt);
    System.out.println(theString);

  7. Step 7

    Short form for experienced ptogrammers:
    String theString = "69";
    int theInt = Integer.parseInt(theString);
    System.out.println(theInt);
    System.out.println(theString);

Tips & Warnings
  • You can only convert Strings which contain numbers for example "33".
  • You cannot convert Stings which contain letters like "AB34X".
  • Strings with hundreds will get rounded. For example "3.4" will become 3.
