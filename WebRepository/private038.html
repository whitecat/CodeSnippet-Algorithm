    Java exposing Private IP to internet

    Submitted by sufinmdq on Sat, 10/17/2009 - 14:30.
    Tags:
    • internet security ip
    • Java Design Patterns
    • Java Q&A
    • Java Security

    Hi to everyone,

    I did buy a D-Link router DIR-600 few days ago and overall it's a good product. So to test it's built-in capabilities I did go to AuditMyPC.com and some other security specialized sites to run a batch of security tests.

    Specifically in AuditMyPC.com was found that -they say this- a bug in Java exposes my internal -that is private- IP to the internet, alongside my current internet address.

    I wonder if there's any way to fix or cloack this? I know it's not a big deal, I run a firewall in my Linux box too and I'm a humble home user, not a corporate one, but it would be cool if I can have my PC fully stealthed.

    Any help greatly appreciated :)
