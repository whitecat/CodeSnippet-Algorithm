Java™ SE 6 Release Notes

Supported System Configurations

JDK Documentation

The configurations listed on this page apply to both the Java SE Development Kit (JDK™) and Java SE Runtime Environment (JRE™) and to Java For Business (JFB). The following table contains links to installation instructions.

Note that some configurations may not be supported on both Java SE and Java for Business (JFB). Refer to the columns in the following table. A blank cell indicates that support is not provided.

Refer to the Supported Locales document for the supported writing systems (languages) for each platform. The JRE does not support all writing systems on all platforms.

Also refer to specific footnotes at the end of this table.

Browser Footnotes

  • Browsers listed above are supported for Java™ Plug-in software and Java™ Web Start software.
    See the Deployment Footnotes for more detail.
  • Firefox 3.6 support was added in 1.6.0_18.
  • Firefox 3.5.1 support was added in 1.6.0_16.
  • Support for IE 8 was introduced in 1.6.0_13.
  • Support for Netscape 4.7.x on Solaris 8 on SPARC and x86
  • Netscape 7 and 7.1 are not officially supported.

Deployment Footnotes

  • For Java Plug-in and Java Web Start on 64-bit Solaris systems, use the 32-bit JRE.
  • For Java Plug-in and Java Web Start on 64-bit Windows systems, the 64-bit JRE is supported as of 1.6.0_12.
  • For Java Plug-in and Java Web Start on 64-bit Linux systems, support is not offered at this time.

Other Operating System Notes

  • Ubuntu 8.04 LTS Desktop Edition support for both JFB and Java SE (x86) in 32-bit was introduced in 1.6.0_18.
  • Red Hat Enterprise Linux 5.4 support for JFB was introduced in 1.6.0_17.
  • Red Hat Enterprise Linux 5.3 support was introduced in JFB 1.6.0_16 and Java SE 1.6.0_18.
  • SLES 11 support was introduced in JFB 6u16-rev-b06 and Java SE 1.6.0_18.
  • Windows 7 support was introduced in 1.6.0_14.
  • Windows Server 2008 R2 support for JFB was introduced in 1.6.0_16.
  • IA32 = Intel Architecture 32-bit
  • x64 processor terminology varies. x64 in the above table could be termed Intel 64, x86-64, AMD 64, EM64T or x64.
  • OpenSolaris 2009.06 support was introduced in 1.6.0_13.
  • OpenSolaris 2008.11 support was introduced in 1.6.0_11.
  • For Itanium support, refer to Java SE Update Release Notes for Itanium .
  • Java SE 6 does not provide support for Windows 98 or Windows ME.
  • OpenSolaris 2008.05 is supported on JDK 6u6p (headless mode only)
  • Red Hat Enterprise Linux 5 support was introduced in 1.6.0_04.

Windows Vista Footnotes

  • Refer to Windows Vista deployment issues on JRE 1.6.0_05 (6u5) and prior releases.
